**Callout** — the documentation admonition block, mapping VitePress `:::tip` / `:::warning` markers into the nxv look. Left accent rail + mono uppercase label + prose.

```jsx
<Callout tone="tip">Use the public web interface at nxv.urandom.io — no install required.</Callout>
<Callout tone="warn" title="HEADS UP">Downloading the index pulls ~220MB.</Callout>
```

Tones: `tip` (Nix blue), `info`, `warn` (amber), `danger` (red).
